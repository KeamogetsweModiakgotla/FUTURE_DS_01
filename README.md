## 📌 Task Overview
  
The objective is to analyse retail sales data and generate meaningful business insights using Excel.

The analysis focuses on:
- Sales performance by category  
- Profit distribution  
- Impact of discounting on profitability  
- Business recommendations based on data insights  

---

## 🛠 Tools Used

- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning Functions (SUM, ISNUMBER, formatting tools)  

---

## 📂 Dataset Features

The dataset includes:
- Sales values  
- Profit values  
- Discount percentages  
- Product categories  
- Order-level transaction data  

---

## 🧹 Data Cleaning Process

The following steps were performed:

- Converted numeric values stored as text into proper numeric format  
- Standardised decimal formatting (comma/percentage corrections)  
- Verified numeric consistency using `ISNUMBER()`  
- Checked and resolved missing/empty values  
- Ensured dataset integrity for analysis

### 2. Profit by Category

---

### 3. Overall Performance
- Total Sales: **R2,295,273.92**
- Total Profit: **R285,988.28**

---

### 4. Discount Impact Analysis

- 0% discount generates the highest profit  
- 10%–20% discounts remain profitable  
- Discounts above 30% result in negative profit  
- High discounts (50%–80%) cause significant losses  

---

## 📈 Key Insights

- Technology is the strongest performing category in both sales and profit  
- Furniture has high sales but extremely low profitability  
- Excessive discounting significantly reduces profit margins  
- The business is profitable overall but sensitive to discount strategies  

---

## 💡 Recommendations

- Limit discounts to below 30% to maintain profitability  
- Improve profit margins in the Furniture category  
- Focus growth strategies on Technology products  
- Continuously monitor discount vs profit relationship  
- Use dashboards for ongoing performance tracking
